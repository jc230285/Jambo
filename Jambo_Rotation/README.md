Jambo_Rot — Minimal WoW Addon

This addon provides a minimap button and a simple options UI.

Files:

- `Jambo_Rot.toc` — TOC file for the addon
- `Core.lua` — Core event handling, saved variables, minimap button creation
- `UI.lua` — Options frame with checkboxes for toggling settings

Slash commands:

- `/jamborot options` — toggle the options UI
- `/jamborot minimap` — toggle minimap visibility

Options (defaults):
- `showOnMinimap`: true
- `autoRotate`: false
- `quickToggle`: true

This is a minimal example to show a map button and simple options. It can be extended with more specific logic and features as needed.
